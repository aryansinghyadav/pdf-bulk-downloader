# PDF Bulk Downloader

A professional-grade React/TypeScript application for downloading PDFs in bulk, designed with speed, stealth, and advanced features for maximum productivity.

## Features

- **Multi-Packet Engine** – Parallel 4-chunk downloading for maximum speed  
- **Stealth Scraping** – Rotates User-Agents and mimics human behavior to bypass blocks  
- **Live Speed Tracking** – Real-time throughput monitoring for all active tasks  
- **Metadata Extraction** – Deep scan for author, title, and creation dates  
- **Advanced Preview** – Full PDF viewer with zoom, page navigation, and fit-to-page controls  
- **Smart Queue** – Concurrent control with pause, resume, and retry logic  
- **Intuitive Workflow** – Clean, user-friendly interface showcasing professional-grade capabilities
